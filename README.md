代码结构：
    /js/common      通用功能层，存放功能职责单一的模块，不涉及业务逻辑
    /js/config      配置层，可进行数据、参数等的配置
    /js/core        一般是核心引导模块，这里放的一个简单实现的模块化加载器
    /js/general     业务层，存放实现具体业务需求的代码
    /js/page        存放单页面所有功能入口的js文件
    /js/plugins     存放第三方的功能，如jqurey等
    /js/template    存放模板文件
    /js/utils       零碎的小功能
